LifeViewer Animations from "Self-Constructions in Conway's Life" talk

Below is a set of nine LifeViewer animations, mostly constructed for a talk given at Gathering for Gardner 13 in Atlanta, Georgia on April 2018. The talk was Self-Construction in Conway's Life. This is a work in progress -- many of the animations could stand to be polished up further.

  • Animation 1: Topic Stack
  • Animation 2: construction arm example. Von Neumann replicator by William R. Buckley, 2007.
  • Animation 3: Intro to Life technology. High-bandwidth telegraph by Jean-Francois Handfield, Feb 2017.
  • Animation 4: construction arm in Conway's Life Prototype universal construction arm by Paul Chapman, 2004. Gemini spaceship by Andrew J. Wade, 2010.
  • Animation 5: Variations on Life construction arms -- Gemini (2010), DOpSearch (2011), spiral growth (2014), Demonoid (2015)
  • Animation 6: Glider Constructions Old and New. Known recipes for c/2 spaceships, plus a cameo appearance by Sir Robin, a knightship discovered shortly before the conference.
  • Animation 7: Slow salvo construction of p456 oscillator. There wasn't time to show this in the talk. This is an extended unidirectional glider construction compiled by Adam P. Goucher's slmake.
  • Animation 8: Four Snark Orientations -- an earlier set of complex unidirectional syntheses, assembled for use in slmake.
  • Animation 9: Sir Robin "synthesis". Constructed on April Fools' Day 2018. Included here only because Sir Robin's cameo appearance was in a "Glider Constructions" animation. A true glider synthesis of Sir Robin isn't likely to be available any time soon.
